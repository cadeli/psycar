# psycar

### A PSYCHEDELIC BACKGROUND ANIMATION JQUERY-PLUGIN


see in action http://ordrumbox.com/cadeli/psycar/

usage :

         $('.container').psycar({
            time_interval_ms  : 100,
            cSpeed            : 0.5 ,
            cAmplitude        : 0.08,  // 0 to 1  
            cDeformation      : 2,
            backColor         : '#FEC657',
            isBezier           : 'true'
               });
